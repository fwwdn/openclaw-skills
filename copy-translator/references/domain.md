# Copy Translator Reference

Use this file when the user wants localization quality rather than a literal translation.

## Translation Modes

- Literal translation: preserve wording closely.
- Natural translation: preserve meaning while smoothing phrasing.
- Localization: adapt tone, idioms, and audience fit for the target market.

## Style Rules

- Preserve intent first, wording second.
- Keep product terms and brand names consistent unless the user says otherwise.
- Avoid obviously machine-translated phrasing.
- If the source is marketing copy, keep it persuasive in the target language.
- If the source is UI copy, keep it short and functional.
