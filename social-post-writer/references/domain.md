# Social Post Writer Reference

Use this file when the user asks for a platform-native social format such as X or Twitter posts, threads, Reddit posts, or short launch copy.

## Platform Notes

- X / Twitter post: short, sharp, one main idea.
- X / Twitter thread: one hook, then a sequenced list of compact posts.
- Reddit post: natural title, context, specifics, and believable tone.
- LinkedIn post: more polished, professional, and lesson-driven.

## Social Writing Rules

- Start with a hook or point of tension.
- Keep each line easy to skim.
- Match the platform's normal tone instead of sounding like ad copy.
- Do not add hashtags unless the user asks.
- For Reddit, avoid exaggerated marketing voice and obvious AI phrasing.
- For X threads, keep each post self-contained enough to read in sequence.
