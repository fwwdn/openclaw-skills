# Longform Writer Reference

Use this file when the user asks for a specific long-form shape rather than a generic article.

## Common Output Shapes

- Essay: strong thesis, 3 to 5 supporting sections, decisive closing.
- Article: clear lead, scannable sections, examples, practical ending.
- Narrative piece: scene-setting opening, chronological flow, reflective close.
- Explainer: define the concept early, walk through examples, end with takeaways.
- Newsletter draft: hook, short intro, 3 to 6 compact sections, closing note or CTA.

## Length Guidance

- `600-900 words`: compact article or short essay.
- `1000-1500 words`: standard long-form post.
- `1800+ words`: deep dive, guide, or feature.

## Writing Rules

- Keep one main argument per section.
- Prefer concrete examples over abstract filler.
- Preserve source facts when source material is provided.
- If the user did not ask for markdown, return clean plain text with normal headings only.
