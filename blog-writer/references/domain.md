# Blog Writer Reference

Use this file when the user wants a blog post with clearer structure, stronger scanning, or better publishability.

## Good Blog Shape

1. Title or angle
2. Lead paragraph that frames the problem
3. 3 to 6 descriptive sections
4. Practical takeaway or closing

## Optional Blog Extras

- SEO-friendly heading wording
- short summary paragraph
- bullet takeaways
- FAQ section
- CTA at the end

## Style Rules

- Make headings informative, not vague.
- Front-load the value in the opening.
- Keep paragraphs readable and concrete.
- Avoid fake stats, fake quotes, and generic hype language.
- If the user provides keywords, weave them in naturally rather than stuffing them.
