# Email Writer Reference

Use this file when the user wants a specific email shape rather than a generic draft.

## Common Email Types

- Follow-up: remind, add context, and end with a light next step.
- Outreach: short intro, clear reason for reaching out, specific ask.
- Reply: acknowledge, answer directly, close clearly.
- Internal update: status first, context second, action items last.
- Apology or recovery: own the issue, explain briefly, propose next step.

## Style Rules

- Keep the first paragraph useful immediately.
- Match formality to the relationship.
- Prefer concrete requests over vague closers.
- Avoid inflated sales language unless the user explicitly wants it.
- If the user asks for concise, keep the draft tight and easy to scan.

## Subject Line Guidance

- Keep it short and specific.
- Reflect the main action or context.
- Avoid clickbait wording for professional email.
