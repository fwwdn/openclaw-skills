# Rewrite Polisher Reference

Use this file when the user wants a specific rewrite style rather than a generic cleanup.

## Common Rewrite Directions

- Clearer: remove ambiguity and make the meaning easier to scan.
- More concise: tighten wording and reduce repetition.
- More professional: sound more polished and structured.
- More natural: reduce stiffness and AI-like phrasing.
- More persuasive: strengthen benefits and momentum without changing the core message too much.

## Style Rules

- Preserve meaning unless the user asks for a stronger rewrite.
- Improve flow and sentence rhythm.
- Remove repetition and unnecessary filler.
- Match the requested tone closely.
