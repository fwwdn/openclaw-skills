# Summarizer Reference

Use this file when the user wants a specific summary shape rather than a generic short version.

## Common Output Shapes

- Bullet summary: 3 to 10 compact bullets.
- Executive summary: short overview, key points, implications.
- Abstract: one short formal paragraph.
- Meeting digest: decisions, updates, next steps.
- TL;DR: one or two dense paragraphs.

## Style Rules

- Keep only the highest-signal information.
- Preserve decisions, claims, action items, and major conclusions.
- Remove repetition and filler.
- Match the requested length and format as closely as possible.
